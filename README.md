# Портфолио Вадима Анохина

Современный двуязычный сайт-портфолио на React, TypeScript, Vite и Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

- `src/content.ts` — контент и переводы `ru/en`.
- `src/App.tsx` — секции лендинга и переключатель языка.
- `src/styles.css` — глобальные стили и Tailwind.
